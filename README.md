# MISTHub

MISTHub is a university-focused collaboration platform developed for the Military Institute of Science and Technology (MIST). The platform aims to create a centralized digital ecosystem where students, faculty members, and alumni can connect, collaborate, and share academic resources.

## Features

### Study Materials

* Upload and share lecture notes, question banks, and academic resources.
* Search and filter materials by department, semester, and type.
* Download and view uploaded files.

### Project Collaboration

* Create project ideas and recruit team members.
* Join ongoing student projects.
* Track participated projects.

### Research Collaboration

* Connect students with faculty research opportunities.
* Express interest in ongoing research projects.
* Promote interdisciplinary collaboration.

### Alumni Network

* Connect with MIST alumni.
* Explore career guidance and mentorship opportunities.
* Build professional relationships within the MIST community.

### User Management

* User registration and authentication.
* Login using username or email.
* Profile management and personalization.

## Technology Stack

* Java
* JavaFX
* FXML
* CSS
* MySQL
* JDBC
* Apache Ant
* NetBeans IDE

## Future Enhancements

* Role-based dashboards (Student, Faculty, Alumni)
* Real-time messaging system
* Event and seminar management
* Internship and job posting portal
* Notification system
* Profile picture support
* Research publication showcase
* Alumni mentorship program

## Vision

MISTHub aims to become the primary academic and professional networking platform for the MIST community by fostering collaboration, knowledge sharing, innovation, and lifelong connections among students, faculty, and alumni.
